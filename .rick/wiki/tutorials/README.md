# Rick CLI 教程系列

> 通过实践教程深入学习 Rick CLI 的各种使用场景

## 📚 教程列表

### Tutorial 1: 使用 Rick 管理简单项目
**难度**: ⭐ 初级
**时长**: 30 分钟
**文件**: [tutorial-1-simple-project.md](./tutorial-1-simple-project.md)

**你将学到**:
- 如何规划和分解任务
- 如何执行任务并处理失败
- 如何积累和复用知识
- 如何使用 Git 管理版本

**适合人群**: Rick CLI 新手，希望快速上手的开发者

---

### Tutorial 2: 使用 Rick 重构 Rick（自我重构）
**难度**: ⭐⭐ 中级
**时长**: 45 分钟
**文件**: [tutorial-2-self-refactor.md](./tutorial-2-self-refactor.md)

**你将学到**:
- 如何使用生产版本规划重构任务
- 如何安装和使用开发版本
- 如何在两个版本之间协作
- 如何验证重构结果并更新生产版本

**适合人群**: 有一定 Rick CLI 使用经验，希望深入理解版本管理机制的开发者

---

### Tutorial 3: 并行版本管理（rick + rick_dev）
**难度**: ⭐⭐ 中级
**时长**: 40 分钟
**文件**: [tutorial-3-parallel-versions.md](./tutorial-3-parallel-versions.md)

**你将学到**:
- 如何同时维护生产版本和开发版本
- 如何在两个版本之间切换和协作
- 如何使用版本隔离避免冲突
- 如何管理多个开发分支

**适合人群**: 需要同时进行多个开发任务的开发者

---

### Tutorial 4: 自定义提示词模板
**难度**: ⭐⭐⭐ 高级

- 优化提示词以提升代码质量
- 创建特定领域的提示词模板

**适合人群**: 希望定制 Rick CLI 行为，优化 AI 编程效果的高级用户

---

### Tutorial 5: 集成到 CI/CD 流程
**难度**: ⭐⭐⭐ 高级
**时长**: 60 分钟
**文件**: [tutorial-5-cicd-integration.md](./tutorial-5-cicd-integration.md)

**你将学到**:
- 如何在 CI/CD 环境中使用 Rick

- 如何集成到 GitHub Actions/GitLab CI
- 如何实现自动化代码审查和测试

**适合人群**: DevOps 工程师，希望将 Rick CLI 集成到自动化流程的开发者

## 🎯 学习路径

### 新手入门路径
```
1. Tutorial 1: 管理简单项目 (30 分钟)
   └─> 掌握基本工作流

2. 阅读 [快速入门](../getting-started.md) (10 分钟)
   └─> 了解常用命令

3. 阅读 [核心概念](../core-concepts.md) (15 分钟)
   └─> 理解 Context Loop

总时长: ~55 分钟
```

### 进阶开发路径
```
1. Tutorial 2: 自我重构 (45 分钟)
   └─> 掌握版本管理

2. Tutorial 3: 并行版本管理 (40 分钟)
   └─> 掌握双版本工作流

3. 阅读 [最佳实践](../best-practices.md) (20 分钟)
   └─> 学习任务分解和依赖设计

总时长: ~105 分钟
```

### 高级定制路径
```
1. Tutorial 4: 自定义提示词 (50 分钟)
   └─> 定制提示词模板

2. Tutorial 5: CI/CD 集成 (60 分钟)
   └─> 集成到自动化流程

3. 阅读 [Prompt Manager 模块](../modules/prompt_manager.md) (15 分钟)
   └─> 深入理解提示词管理

总时长: ~125 分钟
```

## 🚧 常见问题

### Q1: 我应该从哪个教程开始？

**A**: 如果你是新手，从 [Tutorial 1](./tutorial-1-simple-project.md) 开始。如果你已经有一些经验，可以直接跳到感兴趣的教程。

### Q2: 教程中的示例代码可以直接使用吗？

**A**: 是的，所有示例代码都经过测试，可以直接使用。但建议根据你的项目需求进行调整。

### Q3: 完成所有教程需要多长时间？

**A**:
- 新手入门路径: ~1 小时
- 进阶开发路径: ~2 小时
- 高级定制路径: ~2 小时
- 完整学习: ~5 小时

### Q4: 教程中遇到问题怎么办？

**A**:
1. 查看 [故障排查](../getting-started.md#故障排查)
2. 查阅 [常见问题](../faq.md)
3. 提交 [GitHub Issue](https://github.com/anthropics/rick/issues)
